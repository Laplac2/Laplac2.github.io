Linear Quadratic Regulators

理论推导

设线性系统

$$
\dot{x} = Ax + Bu \
y = Cx + Du \tag{1}
$$

其中 $x(t) \in R^n$，$u(t) \in R^m$，初始条件是 $x(0)$，且假设这个系统的所有状态变量都是可测量的。
设状态反馈控制器

$$ u = -Kx \tag{2}$$

代入线性系统方程得

$$ \dot{x} = (A - BK)x = {A_c}x \tag{3}$$

设能量函数

$$ J = \frac{1}{2} \int_0^\infty (x^TQx + u^TRu)dt \tag{4}$$

其中 $Q$ 是半正定矩阵（$Q = Q^T \geq 0$），$R$ 是正定矩阵（$Q = Q^T > 0$）。

  • 方法一

将 $(2)$ 式代入 $(4)$ 得

$$ J = \frac{1}{2} \int_0^\infty x^T(Q + K^TRK)xdt \tag{5}$$

假设存在常量矩阵 $P$ 使得

$$ \frac{d}{dt} (x^TPx) = -x^T(Q + K^TRK)x \tag{6}$$

将 $(6)$ 式代入 $(5)$ 式得

$$ J = -\frac{1}{2} \int_0^\infty \frac{d}{dt} (x^TPx)dt = \frac{1}{2} x^T(0)Px(0) \tag{7}$$

将 $(6)$ 式左边微分展开并代入 $(3)$ 式得

$$
\dot{x}^TPx + x^TP\dot{x} + x^TQx + x^TK^TRKx = 0 \
x^T{A_c}^TPx + x^TPA_cx + x^TQx + x^TK^TRKx = 0 \
x^T({A_c}^TP + PA_c + Q + K^TRK)x = 0 \tag{8}
$$

要使 $(8)$ 式成立，括号里的项必须恒等于 0，即

$$
(A - BK)^TP + P(A - BK) + Q + K^TRK = 0 \
A^TP - K^TB^TP + PA - PBK + Q + K^TRK = 0 \
A^TP + PA + Q + K^TRK - K^TB^TP - PBK = 0 \tag{9}
$$

假设 $K = R^{-1}B^TP$ 代入 $(9)$ 式得

$$
A^TP + PA + Q + (R^{-1}B^TP)^TR(R^{-1}B^TP) - (R^{-1}B^TP)^TB^TP - PB(R^{-1}B^TP) = 0 \
A^TP + PA + Q - PBR^{-1}B^TP = 0 \tag{10}
$$

$(10)$ 式即为代数黎卡提方程（Algebraic Riccati Equation, ARE），根据给定的 $A$，$B$，$Q$，$R$ 解出 $P$ 从而得出 $K$。

  • 方法二

假设存在 $P = P^T$，使

$$
J = x(0)^TPx(0) - x(0)^TPx(0) + \frac{1}{2} \int_0^\infty (x^TQx + u^TRu)dt \
J = x(0)^TPx(0) + \frac{1}{2} \int_0^\infty [\frac{d}{dt}(x^TPx) + x^TQx + u^TRu]dt \
J = x(0)^TPx(0) + \frac{1}{2} \int_0^\infty (\dot{x}^TPx + x^TP\dot{x} + x^TQx + u^TRu)dt \tag{11}
$$

将 $(1)$ 式代入 $(11)$ 式，得

$$
J = x(0)^TPx(0) + \frac{1}{2} \int_0^\infty [(Ax + Bu)^TPx + x^TP(Ax + Bu) + x^TQx + u^TRu]dt \
J = x(0)^TPx(0) + \frac{1}{2} \int_0^\infty (x^TA^TPx + u^TB^TPx + x^TPAx + x^TPBu + x^TQx + u^TRu)dt \
J = x(0)^TPx(0) + \frac{1}{2} \int_0^\infty [x^T(A^TP + PA + Q)x + u^TB^TPx + x^TPBu + u^TRu]dt \tag{12}
$$

已知

$$(u + R^{-1}B^TPx)^TR(u + R^{-1}B^TPx) - x^T(PBR^{-1}B^TP)x = u^TB^TPx + x^TPBu + u^TRu \tag{13}$$

将 $(13)$ 式代入 $(12)$ 式，得

$$J = x(0)^TPx(0) + \ \frac{1}{2} \int_0^\infty [x^T(A^TP + PA + Q - PBR^{-1}B^TP)x + (u + R^{-1}B^TPx)^TR(u + R^{-1}B^TPx)]dt \tag{14}$$

使 $(14)$ 式积分中第二部分等于 0，即 $u = -R^{-1}B^TPx$，结合 $(2)$ 式，得 $K = R^{-1}B^TP$；
使 $(14)$ 式积分中第一部分等于 0，得

$$A^TP + PA + Q - PBR^{-1}B^TP = 0 \tag{15}$$

$(15)$ 式即为代数黎卡提方程（Algebraic Riccati Equation, ARE），根据给定的 $A$，$B$，$Q$，$R$ 解出 $P$ 从而得出 $K$。
