# PlantUML

## VS Code + PlantUML 环境配置

1. 安装 graphviz <http://graphviz.org/>
2. 安装 JDK，配置 JAVA 环境 <https://www.oracle.com/technetwork/java/javase/downloads/index.html>

   注：安装好 JDK 后，记得添加环境变量

3. 在 VS Code 中新建以`.plantuml`为文件名后缀的文件，语法见官网说明文档.
